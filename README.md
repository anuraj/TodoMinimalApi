# Tutorial: Create a Minimal API with ASP.NET Core

Minimal APIs are designed for building HTTP APIs with minimal overhead and dependencies. They are well suited for microservices and lightweight applications that require only the essential files, features, and components in ASP.NET Core. This tutorial covers the fundamentals of creating a Minimal API using ASP.NET Core, following the .NET 10 file-based approach.

This is the File based version of : [Tutorial: Create a Minimal API with ASP.NET Core](https://learn.microsoft.com/en-us/aspnet/core/tutorials/min-web-api?view=aspnetcore-10.0&tabs=visual-studio&WT.mc_id=DT-MVP-5002040)
